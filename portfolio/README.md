# Portfolio Skills

Skills in this folder help with portfolio work, personal knowledge organisation, public project pages, and linked Notion/GitHub references.

## Skills in this category

| Skill | What it does |
|---|---|
| `knowledge-index-updater` | Maintains a clean Notion Knowledge Index across Notion pages and GitHub repos. |

## When to use this category

Use these skills when the task is about:

1. Updating a portfolio page.
2. Organising project references.
3. Connecting Notion pages to GitHub repos.
4. Creating a one-stop index for guides, repos, and references.
5. Keeping public-safe and private items clearly labelled.

## Folder rule

Each skill gets its own folder:

```text
portfolio/<skill-name>/
├── README.md
├── SKILL.md
└── agents/
    └── openai.yaml
```

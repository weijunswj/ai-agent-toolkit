# Guides

Guides are reusable setup and operating instructions for humans and AI agents.

## Sections

- `ai-agent-platforms/`: Codex, Claude Code, ChatGPT web, Claude web, OpenCode, and Antigravity usage notes.
- `mcp/`: Local MCP setup, registry MCP design, installer MCP design, and MCP security.
- `n8n/`: Local n8n setup, upgrades, tunnelling, hosting, workflow sync, and credential safety.
- `cicd/`: Secure CI/CD installer guidance.
- `design/`: UI/UX Pro Max design module and optional local design generator guidance.

Use guides for understanding and setup. Use `templates/` when copying source material into another repo.

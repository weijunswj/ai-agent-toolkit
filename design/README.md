# Design Skills

Skills in this folder help AI agents with frontend UI/UX, design systems, accessibility, responsive layouts, product pages, dashboards, forms, visual QA, and implementation review.

## Skills in this category

| Skill | What it does |
|---|---|
| `ui-ux-secure-frontend-design` | Designs and reviews polished, privacy-safe, security-first web frontends. |

## When to use this category

Use these skills when the task is about:

1. Creating or improving a frontend design system.
2. Planning landing pages, SaaS pages, dashboards, forms, and admin interfaces.
3. Reviewing accessibility, responsiveness, performance, and interaction polish.
4. Checking privacy, safety, and security risks in frontend UI.
5. Turning rough interface ideas into implementation-ready component plans.

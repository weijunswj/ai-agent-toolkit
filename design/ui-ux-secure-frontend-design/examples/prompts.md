# Example Prompts

Use these prompts to trigger the skill in realistic frontend work.

## Create a landing page design system

Use the secure frontend design skill to create a design system for a landing page for a B2B SaaS product. Include colour roles, typography, spacing, component states, accessibility constraints, and privacy notes.

## Review a dashboard

Use the secure frontend design skill to review this analytics dashboard for visual hierarchy, density, responsiveness, accessibility, PII exposure, and maintainability.

## Improve a form

Use the secure frontend design skill to improve this onboarding form. Keep validation intact, make error states clear, avoid dark patterns, and include mobile behavior.

## Build page overrides

Use the secure frontend design skill to create page overrides for a pricing page, a settings page, and an admin audit-log page. Include goals, sections, components, states, mobile behavior, and security/privacy notes.

## Make a SaaS settings page safer

Use the secure frontend design skill to review a SaaS settings page that includes billing, team invitations, API access, notifications, and account deletion. Identify safer UI patterns for high-risk actions.

## Review n8n workflow UI

Use the secure frontend design skill to review an n8n-like workflow editor. Focus on triggers, credentials, executions, webhook URLs, destructive actions, activation controls, and audit-friendly confirmation flows.

## Convert a rough frontend into a polished design

Use the secure frontend design skill to turn this rough React page into a polished, responsive, accessible interface while reusing the existing component library and avoiding new dependencies.

## Check privacy/security risks in a frontend

Use the secure frontend design skill to inspect this frontend for exposed private data, unsafe error messages, analytics/tracking concerns, consent issues, auth/session risks, and dark patterns.

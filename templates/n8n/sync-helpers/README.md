# n8n Sync Helpers

This folder is reserved for review-required n8n workflow sync helper templates.

Use helper assets here only as starting points for a consuming repository after a human reviews the target repo, credentials model, workflow naming, and import/export policy.

Safety rules:

- Do not run live n8n import or export from this toolkit repo.
- Do not store `.env` files, credential exports, credential bindings, private keys, or live workflow exports here.
- Keep helper examples product-neutral and credential-free.
- Treat any future executable helper in this folder as a template asset, not trusted runtime code.

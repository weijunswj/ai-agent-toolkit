# Templates

Templates are review-required source materials for consumer repos.

## Areas

- `agent-rules/`: generated AI-agent rule files and their partials.
- `mcp-configs/`: MCP config examples for Codex, Claude Code, OpenCode, and Antigravity.
- `n8n/`: workflow sync helper templates, sanitizer scripts, and workflow policy notes.
- `cicd/`: secure CI/CD installer prompt and status templates.
- `github-actions/`: safe GitHub Actions patterns for this toolkit.

Do not copy templates blindly. Review target writes and keep secrets out of repo files.

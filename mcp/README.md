# MCP

This folder contains future-facing MCP design documents.

No MCP server implementation is shipped in v1.

## Designs

- `registry-mcp/`: read-only discovery and routing over JSON registries.
- `installer-mcp/`: future approval-gated pack installer.
- `projects/`: safe project-module MCP specs published at root and declared as linked or generated surfaces in `_projects/**/toolkit.project.json`.

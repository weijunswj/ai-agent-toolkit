# Managed App Foundation Review

Project module for the published [Managed App Foundation Review](../../../skills/managed-app-foundation-review/) skill.

The module owns a small first-party planning skill for deciding whether to use low-cost managed providers for auth, user accounts, databases, CRM, forms, email, storage, analytics, and admin workflows instead of building those surfaces from scratch.

See [`_main/`](_main/) for the canonical first-party skill source.

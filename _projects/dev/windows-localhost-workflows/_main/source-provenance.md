# Windows Localhost Workflows Source Provenance

This module was created to adopt the existing standalone `skills/windows-localhost-workflows/` surface into the project-owned source layer.

The migrated material is already AI-facing skill content, so the published skill files are maintained as reviewed curated output under `curated_output_for_ai/` and generated into `skills/windows-localhost-workflows/` by the toolkit sync script.

No product repo code, local logs, `.env` files, credentials, or live service exports are stored in this module.

# Source Manifest: Local n8n Setup

## Preserved In `_main/`

- `README.md`
- `Page 1 - Local Setup.md`
- `Page 2 - Hostinger VPS.md`
- `templates/local-stack/**`

The local stack templates under `_main/templates/local-stack/` are toolkit-authored source for the local `n8n + postgres` Fast Path, with a Compose ngrok service retained as an advanced alternate to the beginner ngrok Docker Desktop extension path.

## AI-Facing Surfaces

- `skills/n8n-local-setup/references/n8n/local-setup.md` and `skills/n8n-local-setup/references/n8n/hostinger-vps.md` are full guide copies generated from `_main/`.
- `skills/n8n-local-setup/references/ai-agent-platforms/codex.md`, `claude-code.md`, `opencode.md`, and `antigravity.md` are skills-first platform routers generated from `curated_output_for_ai/`.
- `skills/n8n-local-setup/templates/local-stack/docker-compose.yml`, `.env.example`, `_n8n-local.cmd`, and `scripts/n8n-local-menu.ps1` are generated by exact copy recipes from `_main/templates/local-stack/`.
- AI-facing skill, n8n/platform indexes, and pack surfaces are generated from `curated_output_for_ai/`.
- Full n8n operating rules are owned by `_projects/development/ai-coding-agent-rules/_main/_partials/n8n-agent-rules.md` and published through `skills/n8n-agent-rules/`.
- `skills/n8n-local-setup/references/n8n-agent-rules.md` is a generated cross-skill reference copied from the canonical `n8n-agent-rules` source for portability. It is labelled as generated cross-skill content and must not be edited directly.
- Generic AI coding agent execution and toolkit skill-routing source are owned by `_projects/development/ai-coding-agent-rules/`; generic templates publish through `skills/ai-coding-agent-rules/`.

## Collapsed Or Retired

- Standalone update, tunnel, Compose appendix, old platform-integration, and generic hosting guide pages were collapsed into the two main pages or skills-first platform routers.
- MCP setup/config source pages and generated config templates were removed from supported routing. MCP setup/config is intentionally not shipped or maintained by this toolkit for now.
- The legacy Windows helper BAT file was removed. `_n8n-local.cmd` is the supported launcher.
- Small compatibility shims for old numbered guide links were removed because the final generated surface now uses clean current paths.

## Excluded

- Credentials, `.env*` except placeholder-only `.env.example`, `.n8n-local/`, `.tmp/`, live n8n exports/imports, package artifacts, and private product files.
- Live workflow state. This toolkit module preserves setup documentation and placeholder-only templates only.
- Full n8n agent operating rules. Use `skills/n8n-agent-rules/` for that contract.

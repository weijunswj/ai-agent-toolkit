The local stack templates under `_main/templates/local-stack/` are toolkit-authored source for the local `n8n + postgres` Fast Path, with the Compose ngrok service as the supported local public tunnel path.

The production Cloudflare stack templates under `_main/templates/production-cloudflare-stack/` are toolkit-authored source for production self-hosting from a local/CGNAT machine through Cloudflare Tunnel. This stack is intentionally separate from the local dev stack and keeps Postgres private, avoids public direct n8n host port `5678`, and uses `cloudflared` with a tunnel token from private environment.

## Collapsed Or Retired

- Standalone update, tunnel, Compose appendix, old platform-integration, Hostinger one-click n8n template guidance, and generic hosting guide pages were collapsed, de-scoped, or replaced by the current main pages and skills-first platform routers.
- [Official n8n Skills](https://github.com/n8n-io/skills) plus instance-level MCP references are secondary skill surfaces. They are not part of the beginner local setup flow.
- The legacy Windows helper BAT file was removed. `_n8n-local.cmd` is the supported launcher.
- Small compatibility shims for old numbered guide links were removed because the final generated surface now uses clean current paths.

## Excluded

- Credentials, `.env*` except placeholder-only `.env.example`, `.n8n-local/`, `.tmp/`, live n8n exports/imports, package artifacts, and private product files.
- Live workflow state. This toolkit module preserves setup documentation and placeholder-only templates only.
- Real Cloudflare tunnel tokens, account IDs, DNS values, domains, IPs, certificates, backups, production `.env` values, and private deployment notes.
- Full n8n agent operating rules. Use `skills/n8n-agent-rules/` for that contract.

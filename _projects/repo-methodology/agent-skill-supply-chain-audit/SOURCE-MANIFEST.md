## Provenance

This module is first-party toolkit-authored source. It was created from the user's requested workflow and this repository's source-of-truth, safety, provenance, and generated-surface rules.

No third-party skill text, scripts, schemas, assets, or generated files were copied into this module. Public GitHub skill repositories reviewed during task discovery were used only as ecosystem context and inspiration, not as source material. Because there is no external upstream file to pin, file-level upstream entries are intentionally absent from [SOURCE-LOCK.json](SOURCE-LOCK.json).

Future maintainers should add active third-party source-lock entries only if this module later copies or adapts material from an external repository.

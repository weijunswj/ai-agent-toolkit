# Knowledge Index Updater Source Provenance

This module was created to adopt the existing standalone `skills/knowledge-index-updater/` surface into the project-owned source layer.

The migrated material is already a complete standalone skill folder, so its canonical source is preserved under `_main/skill/**` and generated into `skills/knowledge-index-updater/` by the toolkit sync script.

No live Notion, GitHub, scheduler, credential, or user data export is stored in this module.

# Development Skills

Skills in this folder help AI coding agents with local development, debugging, dev servers, and project setup workflows.

## Skills in this category

| Skill | What it does |
|---|---|
| `windows-localhost-workflows` | Starts, relaunches, verifies, and troubleshoots localhost development workflows on Windows. |

## When to use this category

Use these skills when the task is about:

1. Starting or restarting a local development server.
2. Debugging localhost issues.
3. Working with package manager startup problems.
4. Fixing port conflicts.
5. Inspecting logs.
6. Running dev workflows from Codex or Claude Code.

## Folder rule

Each skill gets its own folder:

```text
development/<skill-name>/
├── README.md
├── SKILL.md
├── agents/
│   └── openai.yaml
└── references/
    └── optional-supporting-notes.md
```
